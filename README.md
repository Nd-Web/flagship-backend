\# Flagship Backend Project



\## Day 1 Goal

Run a FastAPI backend locally and confirm an endpoint works.



\## Run Locally

1\. Activate venv: `.\\venv\\Scripts\\Activate.ps1`

2\. Install deps: `python -m pip install -r requirements.txt` (later)

3\. Run: `python -m uvicorn main:app --reload`

4\. Test: `GET /health` and visit `/docs`
